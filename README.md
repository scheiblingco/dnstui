# DNSTUI
DNSTUI is a terminal UI for managing DNS records. It provides an easy-to-use interface for adding, editing, and deleting records with various providers.

## Features
- Supports multiple DNS Providers
  - ClouDNS
  - Openprovider
  - Cloudflare
  - Technitium
  - Demo Provider (for testing)
  - If you want another provider added, feel free to open a pull request or an issue.
- Caching of zones and records to minimize API calls and improve performance.
- Input validation for record values and priorities.
- Global search for accounts and zones
- And much more
